# Restart: estudos de Python

Pequeno repositório com scripts simples para praticar lógica de programação em Python.

- `adivinhe.py`: jogo de adivinhação de número entre 1 e 100 com níveis de dificuldade e pontuação que diminui conforme a distância do chute.
- `teste_senha.py`: valida um par usuário/senha fixo, retornando mensagens diferentes para cada caso.

## Pré-requisitos

- Python 3 instalado e disponível no `PATH`.

## Como executar

No terminal, dentro da pasta do projeto:

```bash
python adivinhe.py
python teste_senha.py
```
